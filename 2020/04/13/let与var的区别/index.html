与 var与 let、const的区别


var 与 let、const（常量） 都属于声明变量，先单独比较var与let的区别，
相比let来说，var属于非常古老的语法 它存在许多的缺陷
var：①存在函数作用域和全局作用域 ②提升声明但不提升赋值变量
```

    function sayHi() {
    alert(phrase);

    var phrase = "Hello";
    }

    sayHi(); //并不报错，打印：undefined

   ```

let:拥有块级作用域
const:定义常量，常量：不可更改的变量，常用大写字母命名，可以用在定义颜色，这样即可不记住十六进制颜色还不容易出错，是个很好的点子。

个人总结

我觉得var的出现就是个错误。。。
